Question-1: What are the different ways to select an element in the DOM?

---------------------------------------------------------------------------------------------------------------------------------------------------

we can select elements in the DOM using methods like -->getElementsByTagName(), getElementById(),getElementsByClassName(),querySelector()

Question-2: What is the difference between innerHTML, innerText, and textContent ?

---------------------------------------------------------------------------------------------------------------------------------------------------

innerHtml shows the html inside an element including tags.On the contrary innerText shows only the visible text.textContent shows all text,even hidden ones.

Question-3: What is event delegation in the DOM?

---------------------------------------------------------------------------------------------------------------------------------------------------

Event delegation is a way to add a single event listener to the parent instead of many children.it bubbles up from children to parent.

Question-4: What is event bubbling in the DOM?

---------------------------------------------------------------------------------------------------------------------------------------------------

Event bubbling is a process where an event starts from the target element and moves up through its parent element.parent elements can responds whenever the child elements get triggered.

Question-5: How do you create, add, and remove elements using JavaScript?

---------------------------------------------------------------------------------------------------------------------------------------------------

in case of create: document.createElemet('tagname');
in case of add: parent.appendChild(newElement) or
if we want to add classList then: document.getElementById(id).classList.add(classNames);
in case of remove: parent.removeChild(child);
